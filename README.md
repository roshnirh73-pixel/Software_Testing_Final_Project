# Software Testing Final Project

## Project Overview
This project contains both **Manual and Automation testing artifacts** for a sample application. It demonstrates the complete testing lifecycle, including test planning, test case creation, bug reporting, and automation scripts using Selenium.

---

## Project Structure


---

## Contents Description

- **Automation_Scripts/**: Contains Selenium automation scripts.  
- **Screenshots/**: Screenshots and recordings of test execution and console outputs.  
- **TestCases.xlsx**: Detailed manual test cases for different modules.  
- **Test_Report.pdf**: Final test execution report with results.  
- **Testplan.pdf**: Complete test plan document including scope, objectives, and strategy.  

---

## Tools & Technologies

- **Manual Testing**: Test case creation, bug reporting, test plan preparation  
- **Automation Testing**: Selenium WebDriver, Java  
- **Documentation**: Excel, PDF  

---

## How to Run Automation Scripts

1. Open the `Automation_Scripts` folder in your IDE (Eclipse / IntelliJ).  
2. Make sure Selenium WebDriver is configured in your project.  
3. Run `Test.java` to execute the automation scripts.  
4. Check `Screenshots/` for outputs and recordings.
